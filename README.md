# ☀️ Caffeine News Bot

Ein N8N Telegram Bot der täglich Nachrichten aus verschiedenen Ländern zusammenfasst. Da viele bilingual sind, bietet dieser Bot die Möglichkeit, Nachrichten aus zwei verschiedenen Ländern zu erhalten, um auch Nachrichten aus dem Herkunftsland zu erhalten!

## Features
- Onboarding via Telegram Inline Buttons
- Länderauswahl: 🇩🇪 Deutschland, 🇹🇷 Türkei, 🇫🇷 Frankreich, 🇰🇷 Südkorea
- KI-Zusammenfassung via OpenRouter LLM
- Zweisprachige News (Deutsch + Zweitsprache)

## Tech Stack
- N8N (Self-hosted)
- Telegram Bot API
- RSS Feeds (Tagesschau, Hurriyet, Le Monde, YNA)
- OpenRouter API

## Setup
1. N8N installieren
2. Telegram Bot via @BotFather erstellen
3. OpenRouter API Key holen
4. Workflow JSON importieren
5. Credentials eintragen
